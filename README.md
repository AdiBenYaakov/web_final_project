Trailify - טיולים מותאמים אישית בישראל
Trailify היא אפליקציית אינטרנט מקיפה שנועדה לעזור למשתמשים לגלות ולתכנן טיולי טבע ומסלולי הליכה מותאמים אישית ברחבי ישראל
תכונות מרכזיות
גילוי טיולים דינמי: חיפוש וסינון מסלולים על בסיס העדפות משתמש ונתוני קהילה משותפים.

פרטי טיול אינטראקטיביים: כולל מחשבון מים בזמן אמת, תחזית מזג אוויר ל-7 ימים, ומערכת סימון "מועדפים".

המלצות משתמשים: מערכת Full-Stack המאפשרת למשתמשים להוסיף ביקורות הכוללות דירוג כוכבים, תאריכי ביקור וציוד נדרש.

פרופיל אישי אינטראקטיבי: ניהול טיולים מתוכננים באמצעות יכולת גרירה (jQuery UI Draggable).

מדיה לימודית: מדור טיפים ייעודי הכולל סרטוני וידאו מודרכים למטיילים.

פרויקט מאורגן במבנה תיקיות נקי ולוגי בהתאם לדרישות:
web_final_project/
├── css/            # קבצי עיצוב (index, trips, profile, tips, trip-details)
├── images/         # נכסים ויזואליים ולוגו
├── includes/       # סקריפטים של PHP ודפי HTML משניים
├── js/             # לוגיקה אינטראקטיבית ומימושי jQuery
└── index.html      # דף הנחיתה הראשי
